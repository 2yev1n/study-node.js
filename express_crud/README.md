# node.js express sequelize  MySQl 로 만드는 CRUD

### user 관리

* Create(생성)
  * user 생성 : create
    * id : STRING(15), PK
    * age : INT
    * name : STRING(5)
* Read(읽기)
  * readAll(전체 조회) : findAll
  * readOne(선택 조회) : findOne
* Update(갱신)
  * update(정보 수정) : update
    * age
    * name
* Delete(삭제)
  * delete(id 삭제) : destroy
